- 真·成本价，无续费陷阱

## 二、公司注册方案

### 2.1 方案对比

| 对比项 | 香港公司 | 美国公司 (Stripe Atlas) | 美国公司 (Firstbase) |
|--------|----------|-------------------------|----------------------|
| **注册成本** | ¥3,000-5,000 | $500 (~¥3,600) | $399-599 (~¥3,000-4,300) |
| **年费** | ¥3,000-4,000 | $100 (~¥720) | $100-200 (~¥700-1,400) |
| **注册时间** | 7-14天 | 1-2周 | 1-2周 |
| **税务** | 离岸收入免税 | 需申报但可抵扣 | 需申报但可抵扣 |
| **Stripe友好度** | ⭐⭐⭐⭐⭐ | ⭐⭐⭐⭐⭐ | ⭐⭐⭐⭐⭐ |
| **银行开户** | 相对容易 | Mercury/Firstbase Banking | Mercury/Evolve |
| **维护复杂度** | 低 | 中等 | 中等 |
| **品牌形象** | 国际化 | 美国市场认可度高 | 美国市场认可度高 |

### 2.2 推荐方案：香港公司

**推荐理由**:
1. **Stripe友好**: 香港公司可以无障碍申请Stripe
2. **成本更低**: 综合成本低于美国公司
3. **税务优势**: 离岸收入免税（利润不来自香港）
4. **维护简单**: 年审一次，无需做账报税（无实际经营）
5. **银行开户**: 可选香港本地银行或数字银行（如Airwallex、Wise）

### 2.3 香港公司注册详情

**政府费用**:
- 公司注册处费用: HK$1,545（电子提交）
- 商业登记证: HK$2,200（1年）
- **小计**: HK$3,745 (~¥3,500)

**服务商费用**（含秘书服务+注册地址）:
- 洋骠驹: ¥3,500-4,500
- 恒通商务: ¥3,800-4,800
- FastLane: HK$9,300 (~¥8,500) 全包
- Sleek: HK$8,000-10,000 (~¥7,500-9,300)

**推荐服务商**: 洋骒驹
- 价格实惠（¥3,500起）
- 中文服务
- 包含1年公司秘书
- 包含1年注册地址
- 可协助银行开户

**所需资料**:
1. 董事/股东身份证或护照
2. 住址证明（水电账单/银行账单，近3个月）
3. 公司名称（中英文各一个）
4. 注册资本（默认10,000港币，无需实缴）
5. 经营范围（默认：国际贸易、软件开发等）

### 2.4 美国公司方案（备选）

**Stripe Atlas**:
- 费用: $500注册 + $100/年
- 包含: Delaware C-Corp、EIN税号、$5,000 AWS/Google等积分
- 适合: 计划融资的初创公司
- 链接: https://stripe.com/atlas

**Firstbase**:

## 五、技术实现方案

### 5.1 系统架构

```
用户浏览器 ←→ Vercel (Next.js) ←→ Serverless Functions
                                    ↓
                              Stripe API (支付)
                                    ↓
                              数据库 (Vercel Postgres/Supabase)
                                    ↓
                              视频生成API (Replicate/Fal/阿里云)
```

### 5.2 需要实现的功能模块

1. **用户认证系统**
   - 注册/登录
   - JWT Token管理
   - 密码重置

2. **订阅管理系统**
   - Stripe Checkout
   - 订阅状态同步
   - 续费/取消/升级

3. **权限控制系统**
   - 免费/付费功能限制
   - 生成次数控制
   - 分辨率限制

4. **Webhook处理**
   - payment_succeeded
   - subscription_updated
   - invoice.payment_failed

### 5.3 数据库设计

```sql
-- 用户表
CREATE TABLE users (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  email VARCHAR(255) UNIQUE NOT NULL,
  password_hash VARCHAR(255),
  stripe_customer_id VARCHAR(255),
  subscription_status VARCHAR(50) DEFAULT 'inactive',
  plan VARCHAR(50) DEFAULT 'free',
  credits INTEGER DEFAULT 3,
  created_at TIMESTAMP DEFAULT NOW(),
  updated_at TIMESTAMP DEFAULT NOW()
);

-- 订阅表
CREATE TABLE subscriptions (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  user_id UUID REFERENCES users(id),
  stripe_subscription_id VARCHAR(255),
  plan VARCHAR(50),
  status VARCHAR(50),
  current_period_start TIMESTAMP,
  current_period_end TIMESTAMP,
  cancel_at_period_end BOOLEAN DEFAULT FALSE,
  created_at TIMESTAMP DEFAULT NOW()
);

-- 生成记录表
CREATE TABLE generations (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  user_id UUID REFERENCES users(id),
  prompt TEXT,
  image_url TEXT,
  video_url TEXT,
  status VARCHAR(50),
  cost_credits INTEGER DEFAULT 1,
  created_at TIMESTAMP DEFAULT NOW()
);
```

## 七、注意事项

### 7.1 法律合规

1. **隐私政策**
   - 必须包含数据收集说明
   - Cookie使用声明
   - 用户权利说明

2. **服务条款**
   - 明确服务内容
   - 退款政策
   - 知识产权声明
   - 免责声明

3. **内容合规**
   - AI生成内容版权声明
   - 禁止生成违规内容
   - 用户内容审核机制

### 7.2 税务事项

1. **香港公司**
   - 离岸收入免税
   - 每年仅需年审，无需报税（无实际经营）
   - 保留好Stripe账单作为收入证明

2. **美国公司**
   - 需每年报税（可找会计代理）
   - Delaware Franchise Tax（约$300-400/年）
   - 联邦税申报

### 7.3 风险控制

1. **退款政策**
   - 建议: 7天内无理由退款
   - 已使用的生成次数按比例扣除

2. **欺诈防范**
   - 启用Stripe Radar
   - 设置3D Secure验证
   - 监控异常交易

3. **服务稳定性**
   - 准备降级方案（第三方API故障时）
   - 设置服务状态页面
   - 准备客服响应话术
