# AI视频工坊 🎬

一站式AI视频生成工具平台，包含工具导航 + 内置AI视频生成功能。

## 功能特性

### 📚 AI视频工具大全
- 收录100+主流AI视频生成工具
- 支持搜索、筛选、排序
- 每个工具包含详细评测、价格、评分

### 🚀 内置视频生成（护城河功能）
- **文本生成视频**：输入描述，AI自动生成视频
- **图片生成视频**：上传图片，让图片动起来
- **多模型支持**：Wan 2.1、CogVideoX、Open-Sora
- **参数可调**：分辨率、时长、画面比例

### 💰 付费分享系统
- 付费解锁完整功能
- 生成专属分享链接
- 邀请好友获得佣金

## 技术栈

- **前端**：原生HTML5 + CSS3 + JavaScript
- **部署**：GitHub Pages
- **无需后端**：纯静态网站，可直接部署

## 本地运行

```bash
cd ai-video-hub
# 使用任意静态服务器
python -m http.server 8000
# 或
npx serve .
```

## 部署到GitHub Pages

1. Fork本仓库
2. 进入Settings → Pages
3. Source选择"Deploy from a branch"
4. Branch选择"main"，文件夹选择"/ (root)"
5. 保存后等待部署完成

访问地址：`https://你的用户名.github.io/ai-video-hub/`
